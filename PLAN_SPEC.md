# 맞춤법 키보드 Plan Spec

## ① 한 줄 소개
AI가 실시간으로 맞춤법을 교정해주고 번역까지 되는 안드로이드 키보드 앱

## ② 누가 쓰나요?
- **1차 타겟:** 직장인 (이메일, 보고서, 슬랙), 학생 (과제, 리포트)
- **최종 비전:** 모든 대한민국 사람이 카카오 키보드처럼 기본으로 쓰는 것

## ③ 핵심 기능
1. **실시간 맞춤법 교정** — 문장 완성 시 알림바에 교정 결과 자동 표시
2. **회사 말투 / 격식체 변환** — 토글 하나로 말투 전환
3. **한국어 ↔ 영어 번역** — 키보드에서 바로 번역
4. **두벌식 + 천지인 + 영어 키보드** — 모드 전환 버튼으로 전환
5. **인앱결제** — 1주일 무료 체험 후 월 1,000~2,000원 구독

## ④ 안 만들 것들 (이번 버전)
- 웹사이트
- iOS 버전
- 다국어 지원 (한/영 제외)
- 음성 입력

## ⑤ 디자인 방향
- 미니멀 + 깔끔 (흰 배경, 초록 포인트)
- 네이버 키보드 참고한 레이아웃
- 교정된 부분은 알림바에 자동 표시
