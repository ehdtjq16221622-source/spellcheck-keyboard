무엇을 조절할 수 있나

실제 Android XML과 최대한 같은 축으로 수치를 나눴습니다.

  • 상단 chrome row 높이: toolbar / suggestion / lang / formal
  • 숫자줄 높이, 일반 row 높이, 천지인 row 높이, 기호 row 높이
  • 행별 상하 padding, 좌우 padding, 키 사이 간격
  • 2줄 들여쓰기용 spacer weight, 3줄 shift/delete weight
  • 하단줄 mode / globe / space / enter weight
  • 글자 크기, 보조 글자 크기, 아이콘 크기, radius, shadow
  • 화이트/블랙/핑크/커스텀 테마 색상
