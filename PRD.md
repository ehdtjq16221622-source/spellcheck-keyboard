# 맞춤법 키보드 PRD

## ① 한 줄 소개
AI가 실시간으로 맞춤법을 교정해주는 안드로이드 키보드 앱

## ② 누가 쓰나요?
- **1차 타겟:** 직장인, 학생
- **최종 비전:** 모든 대한민국 사람의 기본 키보드 (카카오 키보드처럼)

## ③ 핵심 기능
1. 실시간 맞춤법 교정 + 빨간 밑줄 표시
2. 회사 말투 / 격식체 변환
3. 교정 횟수 통계
4. 두벌식 + 천지인 키보드
5. 1주일 무료 → 월 구독 인앱결제

## ④ 안 만들 것들
- 웹사이트
- iOS 버전
- 다국어 지원
- 음성 입력

## ⑤ 디자인 방향
- 미니멀 + 깔끔 (흰 배경, 초록 포인트)
- UI 최대한 단순하게
- 틀린 부분은 빨간 밑줄로 명확하게
