Features

단순한 입력 도구가 아니라, 문장을 바로 다듬고 상대에 맞는 톤으로 바꾸는 작업 흐름까지 한 번에 담았습니다.

실시간 교정

맞춤법과 띄어쓰기를 빠르게 다듬어 대화 흐름을 끊지 않도록 돕습니다.

키보드 안 번역

입력 중인 문장을 바로 번역해 메시지 앱을 오갈 필요를 줄였습니다.

말투 변환

회사용, 고객용, 학부모용, 소개팅 톤처럼 상황에 맞는 인상을 빠르게 고를 수 있습니다.

입력 방식 선택

두벌식과 천지인을 지원해 익숙한 입력 습관을 그대로 유지할 수 있습니다.
