입력하신 텍스트는 맞춤법 교정 및 번역 목적으로만 사용되며, 저장하지 않습니다.

수집하는 정보

  • 입력 텍스트 — 맞춤법 교정 또는 번역 요청 시 해당 텍스트

정보 사용 목적

  • 한국어 맞춤법 및 띄어쓰기 교정
  • 한/영/중/일 번역
  • 회사말투 변환

제3자 제공

교정 및 번역 처리를 위해 입력 텍스트가 아래 서비스로 전송됩니다:

  • OpenAI (GPT-4o-mini) — AI 교정·번역 처리
  • Supabase — 서버 인프라 (API 키 보안 관리)

전송된 텍스트는 처리 후 즉시 폐기되며 저장되지 않습니다.

저장 및 보관

  • 입력 텍스트는 서버에 저장하지 않습니다
  • 설정값은 사용자 기기 내에만 저장됩니다
  • 결제 정보는 Google Play에서 직접 처리하며 앱에서 접근하지 않습니다

사용자 권리

  • 앱 삭제 시 기기 내 모든 설정 데이터가 삭제됩니다
  • 맞춤법 교정 버튼을 누르지 않으면 텍스트가 전송되지 않습니다
