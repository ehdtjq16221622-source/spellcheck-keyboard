개인정보처리방침

맞춤법 키보드가 어떤 정보를
어떻게 다루는지 안내합니다.

키보드 앱은 사용자가 입력하는 문장을 다루는 만큼, 어떤 데이터가 서버로 전송되는지와 어떤 데이터가 기기 안에만 남는지 분명하게 설명하는 것이 중요하다고 생각합니다.

최종 업데이트: 2026년 4월 6일

맞춤법 키보드는 교정, 번역, 말투 변환 기능을 실행할 때에만 필요한 입력 텍스트를 처리 서버로 전송합니다. 비밀번호와 같은 민감 입력 필드는 운영체제가 민감 필드로 표시한 경우 전송 대상에서 제외되도록 설계했으며, 사용자가 선택한 커스텀 배경 이미지는 기기 내부에만 저장하고 서버로 업로드하지 않습니다.

Scope

이 방침이 적용되는 범위

본 방침은 안드로이드용 맞춤법 키보드 앱, 앱 내 설정 화면, GitHub Pages로 제공되는 소개 페이지 및 개인정보처리방침 페이지에 적용됩니다.

앱 기능

맞춤법 교정, 번역, 말투 변환, 크레딧 지급 및 차감, 광고 보상 기능에 적용됩니다.

웹 문서

다운로드 안내, 건의사항 링크, 개인정보처리방침 페이지처럼 서비스 소개를 위한 공개 웹 페이지에도 동일한 기준을 적용합니다.

Collected Data

수집하거나 처리하는 정보

서비스 제공에 꼭 필요한 범위에서만 정보를 처리합니다. 기능을 사용하지 않은 일반 입력 자체를 별도로 저장하는 구조는 두지 않았습니다.

기능 요청 시 처리되는 정보

  • 입력 텍스트교정, 번역, 말투 변환을 요청한 문장만 처리 서버와 AI 모델로 전송됩니다. 처리 목적 외 별도 데이터베이스에 저장하지 않습니다.
  • 말투 옵션 및 기능 종류어떤 톤으로 변환할지, 교정인지 번역인지 같은 요청 맥락이 함께 전달됩니다.
  • 광고 보상 요청 정보리워드 광고 완료 여부를 검증하고 중복 지급을 막기 위한 요청 정보가 서버로 전달됩니다.

기기 또는 서버에 저장되는 정보

  • 설정 정보진동, 키 팝업, 입력 방식, 테마, 커스텀 배경 설정은 사용자의 기기 내부에 저장됩니다.
  • 기기 식별자와 크레딧 정보무료 사용량 관리, 광고 보상 중복 방지, 크레딧 잔액 계산을 위해 Android ID 기반 식별자와 현재 크레딧, 마지막 초기화 일자가 Supabase에 저장됩니다.
  • 커스텀 배경 이미지사용자가 직접 선택한 배경 이미지는 기기 저장소에만 보관되며 서버에 업로드하지 않습니다.

Not Collected

수집하지 않거나 제한하는 정보

키보드 앱 특성상 민감한 입력이 섞일 수 있으므로, 아래 정보는 처리 대상에서 제외하거나 별도로 수집하지 않는 방향으로 설계했습니다.

민감 입력 필드

운영체제가 비밀번호, 결제 정보, 기타 민감 필드로 표시한 입력창의 텍스트는 AI 처리 대상으로 전송하지 않도록 제한합니다. 다만 외부 앱이 민감 필드를 올바르게 표시하지 않는 경우 운영체제 한계로 인해 완전한 통제가 어려울 수 있습니다.

연락처와 전체 기기 콘텐츠

연락처, 통화 기록, 메시지 목록, 전체 사진 라이브러리 같은 정보는 수집하지 않습니다. 사진 접근은 커스텀 배경 선택 시 사용자가 직접 고른 이미지에 한해서만 이뤄집니다.

Purpose

정보를 사용하는 목적

문장 처리 기능 제공

맞춤법 및 띄어쓰기 교정, 번역, 상황별 말투 변환 같은 AI 기능을 제공하기 위해 요청 텍스트를 처리합니다.

무료 사용량과 보상 관리

매일 무료 크레딧 제공, 광고 보상 50크레딧 지급, 중복 보상 방지, 잔여 크레딧 계산을 위해 기기 식별자와 크레딧 정보를 사용합니다.

Third Parties

제3자 서비스 이용 및 국외 이전

서비스 제공을 위해 아래 제3자 서비스를 사용합니다. 각 업체의 처리 방식은 해당 업체의 정책을 따릅니다.

OpenAI

  • 이용 목적교정, 번역, 말투 변환을 위한 AI 문장 처리
  • 전달 정보사용자가 요청한 입력 텍스트와 요청 유형
  • 국외 이전AI 처리 인프라 운영 과정에서 국외 이전이 발생할 수 있습니다.

Supabase

  • 이용 목적서버 함수 실행, 요청 중계, 크레딧 정보 저장, 광고 보상 처리
  • 전달 정보기기 식별자, 크레딧 정보, 광고 보상 요청 정보, 기능 요청 데이터
  • 비고서버 비밀값과 API 인프라 관리에 사용됩니다.

Google AdMob

  • 이용 목적리워드 광고 제공 및 측정
  • 전달 정보광고 식별 관련 정보, 기기 정보, 광고 시청 이벤트 정보가 Google 정책에 따라 처리될 수 있습니다.
  • 국외 이전광고 서비스 운영 과정에서 국외 이전이 발생할 수 있습니다.

결제 제공자

  • 이용 목적구독 또는 인앱 결제 처리
  • 전달 정보결제 정보는 Google Play가 직접 처리하며, 앱은 카드번호 같은 민감 결제 정보를 직접 수집하거나 저장하지 않습니다.
  • 비고결제 관련 정책은 Google Play 정책을 따릅니다.

Retention & Security

보관 기간과 보호 조치

보관 기간

  • 입력 텍스트기능 요청 처리 목적에 한해 일시적으로 전송되며, 서비스 운영을 위한 별도 저장소에 보관하지 않습니다.
  • 기기 내 설정앱 삭제 또는 데이터 초기화 전까지 사용자의 기기에 남습니다.
  • 크레딧 정보무료 사용량 제공, 부정 사용 방지, 광고 보상 중복 방지에 필요한 범위에서 보관되며, 삭제 요청 시 검토 후 처리할 수 있습니다.

보호 조치

  • 암호화 전송앱과 서버 간 통신에는 HTTPS/TLS를 사용합니다.
  • 최소 수집 원칙서비스 제공과 운영에 필요한 최소 정보만 처리합니다.
  • 로컬 저장 우선테마, 입력 옵션, 배경 경로 등 민감도가 낮은 설정은 기기 내부 저장을 우선합니다.

Rights

이용자 권리와 문의처

이용자는 앱 삭제 또는 데이터 초기화를 통해 기기 내 설정 정보를 제거할 수 있으며, 서버에 저장된 크레딧 관련 정보 삭제나 개인정보 문의는 아래 연락처로 요청할 수 있습니다.

이용자 권리

  • 삭제 요청서버에 저장된 기기 식별자 및 크레딧 정보 삭제를 요청할 수 있습니다.
  • 아동 보호본 서비스는 만 14세 미만 아동을 대상으로 하지 않으며, 관련 정보를 의도적으로 수집하지 않습니다.
  • 정책 변경본 방침이 중요한 방식으로 변경되면 웹 페이지 또는 앱 내 공지를 통해 안내할 수 있습니다.
